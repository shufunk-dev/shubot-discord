Commands Manager

Manage built-in and custom text commands

  • !ping Check latency of the bot & connection speed.
  • !say <message> Repeats the text provided by the user.
  • !server Displays summary information about the server.
  • !user Displays user profile details and join dates.
  • !help Shows command details and listing menu.
Enable Moderation Module

Allows moderators to kick, ban, and timeout members through the bot.

  • !kick <@user> [reason] Kicks a member (Requires Kick Members permission).
  • !ban <@user> [reason] Bans a member (Requires Ban Members permission).
  • !timeout <@user> <duration> [reason] Timeouts a member (Requires Moderate Members permission; e.g. 10m, 2h).
  • !untimeout <@user> Removes active timeout from a member.
Enable D&D Module

Allows users to roll dice inside Discord channels.

  • !roll [dice] Rolls dice (e.g. d20, 3d6, 1d20+5, or just roll).

Quick Set Up Guide

  1. Go to the Discord Developer Portal.
  2. Create a **New Application** and navigate to the **Bot** tab.
  3. Click **Reset Token** and copy the Token, then paste it in the configuration input above and click **Save**.
  4. Enable **Privileged Gateway Intents**: Check **Presence Intent**, **Server Members Intent**, and **Message Content Intent** (required for messages).
  5. Under **OAuth2 > URL Generator**, check `bot` and `applications.commands` scopes. In Bot Permissions, select **Administrator** or **Send Messages**, **Read Message History**, **Manage Messages**, etc. Copy the URL to invite the bot to your server.
